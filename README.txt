Mission Rana Portfolio Website

Files included:
- index.html
- style.css
- script.js
- Mission_Rana_Resume.pdf

How to publish:
1. Create a GitHub account or sign in.
2. Create a new repository named missionrana.github.io or mission-rana-portfolio.
3. Upload all files from this folder into the repository root.
4. Go to Settings > Pages.
5. Under Build and deployment, select Deploy from a branch.
6. Select main branch and /root folder.
7. Save.
8. Your site will publish at either:
   - https://YOUR-GITHUB-USERNAME.github.io/ if repo name is YOUR-GITHUB-USERNAME.github.io
   - https://YOUR-GITHUB-USERNAME.github.io/mission-rana-portfolio/ if repo name is mission-rana-portfolio

Before publishing:
- Replace placeholder text if needed.
- Add project screenshots later if you want.
- Keep the resume file name as Mission_Rana_Resume.pdf so the website buttons work.
